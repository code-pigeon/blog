在给他人建议时，我们会使用句型

动词た形 + ほうがいいです

- やさいを食べたほうがいいです。（你应该吃蔬菜。）
- しごとに行ったほうがいいです。（你应该去工作。）

当要对某人提出 别做某事的建议时，我们可以使用句型

动词ない形 + ほうがいいです

- それは食べないほうがいいです。（你最好别吃那个。）
- しごとに行かないほうがいいです。（你最好别去工作了。）

我们还可以把疑问助词か加在这些句子的末尾，从而变成“我应该……吗？”或是“我不应该……吗？”

- 行ったほうがいいですか。（我是否应该去？）
- 行かないほうがいいですか。（我不应该去吗？）

当我们要给出一个说明解释的时候，可以将 んです 放在句末。例如，当有人称赞你的日语时，你可以说，

- 実はBusuuで日本語の勉強をしているんです。（其实我一直在博树学习日语。）
- 在这里你用 んです 来分享我们日语说得好的原因。
- じつは東京に住んでいたんです。(其实，我曾住在东京。)

当 んです 与动词一起使用时，我们只需在动词简体句的句末加上 んです 即可。

- 行くんです。（我要去。）
- 行かないんです。（我不去。）
- 行ったんです。（我去了。）
- 行きたいんです。（我想去。）

本とうは行きたいんです。 和 本とうは行きたいです。 两个句子都翻译为“其实，我想去。”

但是，带有 んです 的句子通常用于解释对问题的回答。例如，如果有人问你是否真的想去某处，你可以用 んです 来回答。这个句型表示 “我没有告诉过你（因为当时你反对去那里），但实际上，我很想去。”，而第二个句子只是一个简单的陈述，意思是“其实，我想去。”

- 本とうはおきなわに行きたいんです。（「我没有告诉你，但」其实我想去冲绳。）
- 本とうはおきなわに行きたいです。（ 其实我想去冲绳。）

当 んです 与 i 形容词一起使用时，我们也只用在简体句的句末加上 んです 。

- 優しいんです。（他们很友善。）
- 優しくないんです。（他们不友善。）
- 優しかったんです。（他们以前很友善。）

na 形容词 和 名词 与 んです连接使用时，遵循同样的规则。我们只需在现在时态的简体句的末尾加上 な + んです 。

如果是在过去时态和否定形式的句末，我们则只需要用其简体句接上 んです 就可以了。

- 好きなんです。（我很喜欢。）
- 好きじゃないんです。（我不喜欢它。）
- 先生なんです。（我是老师。）
- 先生だったんです。（我曾是老师。）

んです 也可以用在问句中， んですか。这个问句形式常用于邀请对方针对所发生的状况给出更多的解释和更清楚的说明。它常和どうして（为什么）等词语一起使用。

- あたまがいたいんですか。（你头疼吗？）「请告诉我更多的信息。」
- どうしてこい人とわかれたんですか。（你为什么和恋人分手了呢？）「你得告诉我。」

在回答以 んですか 结尾的问题时，你同样可以使用带有 んです的句子。因为你在告诉他们一些新的信息作为解释！

问题： どうしてかのじょとわかれたんですか。（你为什么和女朋友分手了？）

第一种回答：きのうけんかしたんです。（昨天，我吵了一架。）
第二种回答：きのうけんかしたからです。（因为昨天吵架了。）

どうしたんですか。 这个短语的意思是“发生了什么事？”どうした在简体句中的意思是“发生了什么事？”但当它后面加上了 んですか，就表示“发生了什么事？（请给我一些解释。）”

- どうしたんですか。（发生了什么事？）
- 元気がないですね。どうしたんですか。（你看起来很疲惫。发生了什么事？）

つごうがわるい / 都合が悪い （tsugou ga warui）的意思是“ 不方便”。然而，当它用于谈话时，意思是“我办不到。”

- それはつごうがわるいです。（那对我来说有些困难。）
- 日曜日はつごうがわるいです。（我星期天不太方便。）

仕方がない / しかたがない （shikata ga nai）的意思是”没办法了。”所以，如果某人拒绝了请求，我们经常这样回应。
我们还可以在句末加上ね，表现得更为随意或是友好，可以说： しかたがないですね。 我们也可以去掉助词说： しかたないですね。

- しかたがないです。（没办法。）
- それはしかたがないですね。（那也没办法。）
- しかたないですね。（真没办法。\[更随意\]）

しょうがない （shouganai）是另一个在日本常用的短语，它是しかたがない的口语形式。两者的意思是一样的，但这个短语在对话中的用法更为随意。

- しょうがないです。（没办法。）
- それはしょうがないですね。（那也没办法。）

当别人询问你 大じょうぶですか。 你可以这样回答： 大じょうぶです。（我没问题。） 或 大じょうぶじゃないです。/ 大じょうぶではありません。（我感觉不太好。）

- ねつはないから、大じょうぶです。（我没有发烧，所以没问题。）
- 今日は休みだから、大じょうぶですよ。（今天我放假，所以没问题。）
- しごとがいそがしいから、大じょうぶではありません。（因为工作很忙，所以我不行。）


如果有人问你 おげん気ですか。 我们可以简单地回答 はい、げん気です。 或者 いいえ、げん気じゃないです。 / いいえ、げん気ではありません。.

你还可以在回答中加入原因。来看看怎么用！

- はい、8時かんねたからげん気です。（是的，因为我睡了 8 个小时，所以我精神很好。）
- いいえ、今日はテストがあるから、げん気じゃないです。（不太好，因为今天有考试，所以我感觉不太好。）

助词有时会接在 疑问词+か 后面，但它是具有选择性的！需要注意的是，有些句型必须添加助词！
⚠️いつ不需要接助词，因此在いつか后面无需再添加助词。

- 何か(を)食べたいです。（我想吃一些东西。）
- だれか(が)いますか。（有人在这儿吗？）
- だれかと電話していましたか。（你在和谁打电话呢？）
- 来週どこか(に)行きませんか。（我们下周何不去个什么地方？）
- どこかで会いましたね。（我们曾经在哪里见过，不是吗？）
- いつかけっこんします。（我总有一天会结婚。）

いつか？五日（いつか）？

日语单词“某一天”和“5日（天）”的平假名是一样的，但在口语中，语调是不同的。

いつか：重音在い上。
五日：重音在つ上，か与つ保持相同的音高。

在简体句中，我们时常用升调的 の 表示提问。在以名词或na形容词结尾的句子中，我们会使用 なの。

- あした学校に行くの？（你明天会去学校吗？）
- どうしてスキーが上手なの？（为什么你那么擅长滑雪？）


な 常用于句子末尾，表达自己的情感。这种用法可以表示愿望。所以你甚至可以用它来自言自语！😂

- どうぶつの中でねこが一番かわいいな。（在所有的动物中猫最可爱！）
- このまちはいつもにぎやかだな。（这个城市总是很热闹！）
- すしよりラーメンのほうが食べたいな。（比起寿司，我更想吃拉面！）

新しい日本語です！ ✍️

动词 なる 的意思是“成为/变成”。我们将它与形容词组合使用，来表达某人或某事成为了该形容词所指的状态。

让我们来看一些例句！

- 気候が暑くなりました。（天气变热了。）
- 漢字が好きになりました。（我开始喜欢汉字了。）

当我们将 なる 与 i形容词连接使用时，要遵循下列规则：
去掉形容词的词尾 い ，并添加 く+なる。

- 暑い → 暑くなる （变热）
- 可愛い → 可愛くなる （变可爱）
- 楽しい → 楽しくなる （变得有趣）

当我们将 なる 与 na形容词连接使用时，要遵循下列规则：在na形容词的后面
添加 に + なる 。

- 好き → 好きになる （变得喜欢）
- 下手 → 下手になる （变得不擅长）
- にぎやか → にぎやかになる （变得热闹）

要注意一些例外情况！
いい 意为“好”，当它与动词 なる连接使用时，要变成 よくなる。这也同样适用于形容词かっこいい。

- いい → よくなる （变好）
- かっこいい → かっこよくなる （变酷/变帅）


当 なる 与 名词连接使用时，你要遵循与na形容词相同的规则：在名词的后面
添加 に+なる ！

- 先生になる （成为一名老师）
- 大学生になる （成为一名大学生）
- 友だちになる （成为朋友）

当我们在谈论我们想成为什么时，会使用 なりたい。来看一些例句！

- 日本語の先生になりたいです。（我想成为一名日语老师。）
- あの人と友だちになりたい。（我想和那个人成为朋友。）


## 指出问题
现在我们使用同样的句型，只是换成形容词！我们只需要在 i 形容词 后面 加上こと ，在 na 形容词后面 加上なこと 。

当你想指出问题所在时，这个句型很有用！一起来看看。

- 問題は手が小さいことです。（问题是我的手很小。）
- 問題は日本語が難しいことです。（问题是日语很难。）


## 比较
在比较两个事物时，我们使用如下句型：
第1项事物 のほうが 第2项事物 より 形容词 です。
在这个句型中，第1项事物是 我们正在讨论的，第2项事物是 我们正在与第1项事物进行比较的。

- 日本のほうがロシアより小さいです。（日本比俄罗斯小。）
- とうきょうのほうがおおさかよりおおきいです。（东京比大阪大。）


在日语中，只要正确使用助词，词语的顺序是可以变换的。这意味着我们可以调换我们刚才所学句型中的语序，只要确保 のほうが 位于我们谈论的话题后面， より 位于被比较的事项之后即可。

第2项事物 より 第1项事物 のほうが 形容词 です。

- おちゃのほうがコーヒーより好きです。（比起咖啡，我更喜欢茶。）
- コーヒーよりおちゃのほうが好きです。（比起咖啡，我更喜欢茶。）

## 询问别人偏好
在针对两项事物的比较向别人提出问题时，我们使用疑问词 どちら （哪个）。句型如下：
选项1 和 选项2 と＋どちらのほうが 形容词 ですか。

- 日本語とえい語とどちらのほうがむずかしいですか。（日语和英语哪个更难？）
- 日本語のほうがえい語よりむずかしいです。（日语比英语更难。）

在非正式的对话中，我们用 どっち 代替 どちら （哪个）使用。意思是一样的！

- バナナといちごとどっちのほうが好き？（香蕉和草莓，你更喜欢哪个？）
- いちごのほうが好きだよ！（我更喜欢草莓！）


## 最高级

**どちら——两个选项，どれ——三个及以上选项**

如果我们想突出某个事物在一个群组中是最好的，我们会使用如下句型：
群组+のなかで+名词+が+いちばん+形容词+です。

- 果物の中で林檎が一番好きです。（在所有的水果中，我最喜欢苹果。）
- 季節の中で夏が一番嫌いです。（在所有的季节中，我最不喜欢夏天。）
- 家族の中で母が一番優しいです。（在所有的家庭成员中，妈妈最温柔。）

当询问别人在3个或以上选项中最喜欢哪一个时，我们会使用疑问词 どれ (dore) “哪一个”。
例如，我们可以提出问题“在所有的 动漫中，你最 喜欢 哪一部？ ”

- アニメの中でどれが一番好きですか。（在所有的动漫中，你最喜欢哪一部？）
- 果物の中でどれが一番美味しいですか。（在所有的水果中，你觉得哪一种最美味？）
- 季節の中でどれが一番寒いですか。（在所有的季节中，哪个季节最寒冷？）

どれ 的意思是“哪一个（3个及以上选项）”，但是当你谈论的对象是人时，你应该使用 だれ （谁）。

- かぞくの中でだれが一ばんやさしいですか。（在所有的家庭成员中，谁最温柔？）
- キャラクターの中でだれが一ばんかっこいいですか。（在所有的角色中，谁最酷？）
- 先生の中でだれが一ばんこわいですか。（在所有的老师中，谁最凶？）



## 形容词表原因
我们可以用形容词来解释为什么我们喜欢某些事物。我们常在から前使用普通形。
な形容词接「から」时，需要带上它的词尾「だ」，变成「だから」。

- 私の日本語の先生はやさしくて、あたまがいいから、とてもすきです。（我的日语老师很温柔，还很聪明，我非常喜欢她！）
- きょうとはふるくて、きれいだから、すごく好きです。（京都古老而美丽，我非常喜欢。）



## 连接形容词
现在让我们来学习如何连接形容词！为此，我们需要将第一个形容词变为 て形。
对于i形容词， 我们将去掉词尾的 い 并添加 くて： やさしい - やさしくて
对于na形容词，我们就在其后加上 で： かんたん - かんたんで


我们使用て形来连接的形容词，要么全都带有正面含义，要么全都带有负面含义。当句中一个形容词带有正面含义，另一个形容词带有负面含义时，我们就用 が （但是）来连接它们。

- 日本語は楽しくて、面白いですが、難しいです。- 日语好玩又有趣，但是很难。

## 季节
在谈论季节的时候，我们可以使用に和は。

に 是表示时间的助词，常用于表达在这个季节里做了什么，或是准备做什么。

は 的用法更为广泛，它也可以用在和に相同的句型中。

- はるにおはなみをします。（在春天，我会“赏花”。）
- はるはおはなみをします。（（一般而言）在春天，我会“赏花”。）
- 三月に日本に行きます。（三月，我会去日本。）
- 三月は日本に行きます。（（一般而言）三月我都会去日本。）


## 天气

| そら / 空 / sora		| 天空 		|
| あめ / 雨 / ame 		| 下雨 		|
| はれ / 晴れ / hare		| 晴天/晴朗	|
| くもり / 曇り / kumori	| 多云 		|
| ゆき / 雪 / yuki 		| 下雪 		|


在表达下雨或下雪时，我们可以使用动词 ふる/降る （下，降下）。例如， あめがふります。 字面意思是“正在下雨”，或换句话说，下雨了。

我们可以使用动词 やむ/止む / yamu（停止）来表达雨停了。

- 今日は雪が降ります。（今天会下雪。）
- 昨日から雨が降っています。（从昨天开始一直在下雨。）
- 雨はいつ止みますか。（雨什么时候停？）

## 可能

我们用 でしょう （deshou）来表达“可能”。我们可以把它与名词或形容词一起放入下列句型：
名词/形容词+でしょう。

但是，这是一个非常正式的短语，常用于天气预报或新闻广播。

- 今日は雨でしょう。（今天可能会下雨。）
- あしたの天気ははれでしょう。（明天的天气可能是晴天。）
- あのすしはおいしいでしょう。（那寿司应该很好吃。）
- かのじょはえが上手でしょう。（她可能很擅长画画。）

我们还可以将 でしょう 与动词一起使用。来看看怎么用！
动词（普通形）+ でしょう。

- 今日は雨がふるでしょう。（今天可能会下雨。）
- 今日はしぶやに行くでしょう。（我今天可能会去涩谷。）
- 八時までゆきがやまないでしょう。（可能到8点雪都不会停。）

词语 たぶん/多分 （tabun）也有“可能”的意思！我们可以将它与 でしょう 搭配使用，表示强调。

⚠️带有 たぶん 的句子在日语口语中更为常见。

- 今日はたぶん雨でしょう。（今天可能会下雨。）
- あのレストランはたぶんおいしいでしょう。（那家餐厅应该不错。）
- 今日はたぶんしんじゅくに行くでしょう。（我今天应该会去新宿。）

たぶん 也可以不和 でしょう一起搭配使用，它的意思不变。

- 母はたぶんうちにいます。（妈妈可能在家。）
- たぶん雨がふります。（可能会下雨。）
- かれはたぶん来ません。（他可能不会来。）

我们在表达“可能”时使用下面句型：
名词/形容词/动词（普通形）+ かもしれません。
来看一些例句！

- 今日はゆきがふるかもしれません。（今天可能会下雪。）
- あしたは学校に行かないかもしれません。（我明天可能不会去学校。）
- 私はかれが好きじゃないかもしれません。（我也许不喜欢他。）
- あの人は学生かもしれません。（那个人也许是个学生。）

でしょう 和 かもしれません 在大多数情况下可以互换，但是かもしれません所表达的不确定语气更强。

- かさはたぶんうちにあるでしょう。（雨伞可能是在家里。）
- かさはうちにあるかもしれません。（雨伞也许在家里吧。）
- あの人はたぶん学生でしょう。（那个人应该是个学生。）
- あの人は学生かもしれません。（那个人也许是个学生。）

 かもしれません 的简体形式是 かもしれない！

- 日本に行くかもしれないから、日本語のべんきょうをします。（因为我有可能去日本，所以我要学习日语。）

在朋友或家人之间的十分随意的对话场合，可以用省略形式 かも 代替かもしれません使用。

⚠️ 当然，这样的形式不能像 かもしれない那样活用，所以它无法与 から一起使用！来看一些例句。

- 今日は雨がふるかも。（今天可能会下雨。）
- 先生は来ないかも。（老师可能不会来。）

## 时态
### 持续动作
我们学过用“て 形 + いる”来形容一个正在进行的动作或从过去某个特定时间点开始进行的动作。

要表达从什么时候开始某个动作，我们用 から （开始）。来看一看这个句型：
（动作开始的时间） から + て 形 + います。

- 先月から日本語を勉強しています。 
- いつからサッカーをしていますか。（你是从什么时候开始踢足球的？）
- 一年前からギターを弾いています。（我从一年前开始弹吉他。）

要表达过去一直持续做某件事，可以把 います 换成 いました。 ⏰

- 前、日本で仕事をしていました。

要表达某动作在某个时间点结束，我们可以用 まで （至；直到）

让我们看一看这个句型：
（结束的时间点） まで + ていました。

先月まで、美容院で仕事をしていました。

### 习惯性的行为
て 形 + いる 也用来描述我们持续性或习惯性的行为。 
当 て 形 + いる 用在这样的语境中时，我们常会用诸如 よく / yoku （时常）和 毎日 / mainichi （每天）。

- 毎日日本語を勉強しています。
- リサさんはよく<ruby>家<rt>うち</rt></ruby>で仕事をしています。(丽莎经常在家办公。)
- 私は毎日自転車で学校に通っています。（我每天骑自行车上学（和放学）。）


在谈论过去的习惯时，把 います 换成 いました。 ⏰

- 私は前、毎日タバコを吸っていました。（我以前每天都吸烟。）
- 毎日図書館に通っていました。

因为“て形 +います”表示习惯性活动，所以你可以用以下问题来询问他人的职业：

- 何をしていますか。（你（的工作）是做什么？）

### 曾经做过某事
在表达我们曾经做过某事时，我们使用句型：
动词た形+ ことがあります。

- このえいがを見たことがあります。（我（以前）看过这部电影。）
- 日本に行ったことがあります。（我（以前）去过日本。）

在表达我们以前从未做过某事时，使用以下句型：
动词た形 + ことがありません。

- ラーメンは食べたことがありません。（我没有吃过拉面。）
- 北海道 (ほっかいどう)に行ったことがありません。（我没有去过北海道。）

我们可以用ないです来表达我们从未做过某事。

ないです是简体形（ない）和敬体形（です）的组合，所以尽管这种表达也很有礼貌，但它的语气与ことがありません相比，更加随意。我们可以在面对某些比我们年长或资格较老、但与我们关系比较亲近的对象时，使用这个句型。

- ラーメンは食べたことがないです。（我以前从没吃过拉面。）
- テニスをしたことがないです。（我以前从没打过网球。）

在简体句中，我们可以省掉助词。例如，我们可以使用“た形ことある”代替“た形ことがある”。例如： 日本に行ったことある？ （你去过日本吗？）

- 敬体： これを食べたことがありますか。（你吃过这个吗？）
- 简体： これ食べたことある？（你吃过这个吗？）
- 敬体： いいえ、食べたことがありません。（没有，我没吃过（这个）。）
- 简体： ううん、食べたことない。（没有，我没吃过（这个）。）


## 动词

1. RU 动词：以 -iru 和 -eru 发音结尾的动词，例如，みる (miru)、たべる (taberu)，但是有一个例外 かえる （帰る）
	- 首先来看 RU 动词（以 -iru 和 -eru 结尾的动词）。我们只需要删除词尾的る (ru)，然后加上ます即可。
	- 食べる　→　食べます。　見る　→　見ます。
2. U 动词：其他所有动词，例如，のむ (nomu)、いく (iku)、よむ (yomu)
	- 要构成 U 动词（所有不以 -iru/-eru 结尾的动词）的ます形，我们需要将词尾的“u”音变为“i”音！
	- 行く　→　行きます。　読む　→　読みます。　飲む　→　飲みます。　話す　→　話します。　遊ぶ　→　遊びます。　作る　→　作ります。
3. 不规则动词：不规则动词只有 2 个！分别是する (suru) 和くる (kuru)
	- する → します。　くる　→　きます（来，来到）。

日语的三类动词：

| 类别 | 又称      | 识别特征（辞书形）                                 | 例子         | 否定形（ます形） |
|-----|-----------|--------------------------------------------------|--------------|----------------|
| 一类 | 五段动词  | 词尾在 う、く、ぐ、す、つ、ぬ、ぶ、む、る 这9个段上           | 書く、話す、買う | き→かない        |
| 二类 | 一段动词  | 词尾一定是 ~る，并且 ~る 前面的假名在 い段 或 え段 上 | 食べる、起きる  | べ→べない        |
| 三类 | 不规则动词 | 只有两个：する 和 来る                             | する、来る     | する→しない       |

一类动词（五段动词）

“五段”的含义：之所以叫“五段动词”，是因为其词尾的发音会在あ、い、う、え、お五段中变化。例如动词「書く」：

- 未然形：書かない（否定）
- 連用形：書きます（礼貌体）
- 終止形：書く。（结句）
- 連体形：書く時（修饰体言）
- 仮定形：書けば（假定）
- 命令形：書け！（命令）

最重要规则：它的ます形变法是：将词尾的「う段」假名变为同行「い段」假名 + ます。

- 書く → 書きます
- 話す → 話します
- 立つ → 立ちます
- 遊ぶ → 遊びます
- 買う → 買います

注意：有一部分动词虽然以「~る」结尾，但它们是一类动词，而不是二类。例如：「切る」（きる）、「知る」（しる）、「入る」（はいる）。这些需要特别记忆。

二类动词（一段动词）

“一段”的含义：因为其词尾变化只在一个段上（い段或え段）进行，所以叫“一段动词”。

核心规则：它的ます形变法非常简单：去掉词尾的「る」直接加「ます」。

- 食べる → 食べます
- 起きる → 起きます
- 見る → 見ます
- 寝る → 寝ます

识别技巧：如果一个动词以「~る」结尾，并且「~る」前面的假名是「い段」（き、し、ち、に…）或「え段」（け、せ、て、ね…），那么它极大概率是二类动词。

三类动词（不规则动词）

它们不遵循任何规则，必须单独记忆。

「する」（做）：

- ます形：します
- 否定形：しない
- 很多“名词+する”构成的动词（如：勉強する、料理する）都属于这一类。

「来る」（来）：

- ます形：きます（注意发音是「き」而不是「く」）
- 否定形：こない

**学习建议与记忆口诀**

判断流程：

1. 看到动词，先看是不是 「する」 或 「来る」？是 → 三类。
2. 如果不是，看它是不是以 「る」 结尾？不是 → 一类。
3. 如果是以「る」结尾，看「る」前面的假名是不是在 「い段」 或 「え段」 上？
	- 是 → 二类（但要小心「切る」「知る」等一类动词例外）。
	- 不是 → 一类。

实用口诀：

“去ます，看词尾，定类别。”

- 拿到「書きます」，去掉「ます」得到「書き」。
- 词尾是「き」，属于「かきくけこ」行，所以它是一类动词。

## 形容词
**i 形容词**

この映画は面白いです。(这部电影很有意思。)

この映画は面白くないです。(这部电影没意思。)

昨日の映画は面白かったです。(昨天的电影很有意思。)

昨日の映画は面白くなかったです。(昨天的电影没意思。)

**na 形容词**

私は今元気です。(我现在很好。)

私は今元気じゃないです。(我现在不太好。)

私は昨日元気でした。(我昨天很好。)

私は昨日元気じゃなかったです。(我昨天不太好。)

---

“i 形容词です”的过去时态：词尾的 い 变成 かった。

- この映画は面白いです。
- 昨日の映画は面白かったです。

“i 形容词”的过去否定形式：把 くない 变成 くなかった。

- この映画は面白くないです。
- 昨日の映画は面白くなかったです。


私は先月ほっかいどうに行きました。さむくなかったです。（我上个月去了北海道，天气不冷。）

いい （很好）
よくない （不好）
よかった （之前很好）
よくなかった （之前不好）

い形容词（直接接名词）

- 詰まらない（无聊的）
- 素晴らしい（极好的）
- 凄い（绝妙的）
- 面白い（有趣的）
- 古い / ふるい（古老的）
- 嬉しい / うれしい（开心的）

な形容词（な+名词）

- 有名
	- 有名な人
- 変（古怪的）
	- 恵子さんは変な人です。
	- 変な店。
- 好き
	- 好きな音楽。
- 元気
	- 元気な女の子。

“じゃないです”的过去时态：要改为过去时态，可以把 い 换成かった： じゃなかったです！

- さきさんは去年会社員じゃなかったです。（早纪小姐去年不是一名公司员工。）
- 私はきのう元気じゃなかったです。


要让 “（名词/ na 形容词） じゃなかったです”的表达方式更有礼貌，可以做出如下调整：
じゃ → では
ないです → ありませんでした。

来看一看以下句子，这些句子都表示“我身体感觉不太好”。🤒

| 更随意   | げんきじゃなかったです。   |
|----------|---------------------|
| ⇅       | げんきじゃありませんでした。 |
| ⇅       | げんきではなかったです。   |
| 更加正式 | げんきではありませんでした。 |


当我们将から 与na形容词连用时，可以用以下句型：
na形容词 + だ + から、

- ふじ山はきれいだから、しゃしんをとりたいです。（富士山很美，所以我想去摄影。）
- この本が好きだから、ほしいです。（我喜欢这本书，所以很想要一本。)


## 关联
### 表转折
你还记得でも (demo) 吗？它放在句首，与前面的句子形成矛盾关系。
但我们也可以用一个句子来表达矛盾！为此，我们使用以下结构：
第一个表述 + が + 、 + 第二个表述。
不要忘记在が (ga) 后加上标点符号！

- このおかしはおいしいです。でも、たかいです。/ Kono okashi wa oishii desu. Demo, takai desu.（这种甜点很好吃。但是很贵。）
- このおかしはおいしいですが、たかいです。/ Kono okashi wa oishii desu ga, takai desu.（这种甜点很好吃，但是很贵。）
- 学校に行きませんでしたが、うちで勉強しました。（（我）没去上学，但（我）在家学习了。）
- このケーキは安いですが、美味しいです。（蛋糕虽然便宜，但很好吃）

### 表达“但是”的新方法！

我们常在敬体句中使用 が 把句子分为两部分，来表达“但是”的含义。但是在 简体句 中，你经常会听到 けど。在这种情况下，我们并不会在名词和na形容词后面去掉 だ 。一起来完成下列敬体句和简体句！

|		|																			|
|-------|---------------------------------------------------------------------------|
|礼貌	| 私は日本の出しんですが、おんせんに行ったことがありません。（我是日本人，但我从没去过温泉。）	|
|随意 	| 私は日本の出しんだけど、おんせんに行ったことがない。（我是日本人，但我从没去过温泉。）		|
|礼貌	| 私はさかなを食べますが、にくを食べません。（我吃鱼，但不吃肉。）						|
|随意 	| 私はさかなを食べるけど、にくを食べない。（我吃鱼，但不吃肉。）							|

### 表原因
我们还可以构成包含两部分的句子来解释我们为什么做某事。首先说明做这件事的原因，然后说明正在做的事情。这种句型采用以下结构：
原因 + から + 、 + 说明。
不要忘记から (kara) 后的标点符号！我们来看一些例子。

- 来年日本に行きますから、日本語を勉強します。/ Rainen Nihon ni ikimasu kara, nihongo o benkyou shimasu.（因为我明年要去日本，所以我学日语。）
- 寒かったですから、公園に行きませんでした。/ Samukatta desu kara, kouen ni ikimasendeshita.（因为天冷，所以我没有去公园。）

如果我们只想单独列举某事发生的原因（我喜欢寿司），可以使用以下句型：
动词或形容词的普通形から + です。 也可以说： すしがすきだからです。

- 好きだからです。（因为我喜欢它。）
- いそがしかったからです。（因为我很忙。）
- ほしいからです。（因为我想要它。）


现在你学到了如何用不同的方式去阐释原因！

### 用に还是で？
に 在 ある / いる 的句型里表示 存在的场所。
に 在接 行く 等动作动词时，表示 前往的地点/目的地。
で 在接 食べる 等动作名词时，表示 动作发生的场所。

- 私は日本にいます。/ Watashi wa Nihon ni imasu.（我在日本。）
- 私は日本に行きます。/ Watashi wa Nihon ni ikimasu.（我将去日本。）
- 私は日本ですもうを見ます。/ Watashi wa Nihon de sumo o mimasu.（我将看日本的相扑比赛。）

## 表愿望
### 否定式
在表达我们不想要什么的时候，我们会使用 欲しくない / ほしくない，因为 ほしい 是i形容词。
我们使用以下句子结构：
名词+は+ ほしくないです。。

⚠️注意！在使用ほしくない时，我们常用 は 代替 が！

- かばんは欲しくないです。（我不想要包包。）
- このコーヒーはほしくないです。（我不想要这个咖啡。）
- 新しいスーツケースはほしくないです。（我不想要新的行李箱。）


したくないです ✋

在表达不想做某事时，我们使用动词的たい形，并把たい换成たくない。

|						|		|
|-----------------------|-------|
|かいたい → かいたくない 		| 不想买	|
|食べたい → 食べたくない 	| 不想吃	|
|したい → したくない 		| 不想做	|

和ほしくない一样，动词的たくない形也要使用助词 は。
名词+ は + 动词たくない形 + です。

- スーツケースはかいたくないです。（我不想买行李箱。）
- ケーキは食べたくないです。（我不想吃蛋糕。）


### 过去式
你有没有注意到，动词后续たい的活用方法与 i形容词相同？它们在过去时态中也相同！

我们只需要 去掉い并 加上かった就可以表达我们过去曾经想做什么事。
在谈论我们过去不想做的事时，就去掉い 并 加上くなかった。

食べたかった（我之前很想吃。）

食べたくなかった（我之前不想吃。）


### 要点小结！
在之前的课程中，我们学习了如何将助词 が 用于 たい形， 将助词は 用于 たくない形。
然而，当我们在谈论我们想去哪里时，则常在行く / いく（去）前使用助词に或へ 。

日本に行きたいです。（我想去日本。）

学校に行きたくないです。（我不想去学校。）

---

在使用句型 が+动词たいです时，我们可以用助词 を代替が。
它们的意思相同，可以互换使用，适用于大部分名词。

ゲームがしたいです。（我想打游戏。）

ゲームをしたいです。（我想打游戏。）


---

我们一直用て形来表明动作的先后顺序。这意味着我们不会再做其他事。
然而，当要列举出没有明确先后顺序或没有特别限制的活动，我们就会使用下列句型：
动词た形 + り、动词た形 + りします。。我们来比较下面的例句：

- 日曜日はレストランに行って、えいがを見ます。（星期天，我要去餐厅，然后去看电影。）
- 日曜日はレストランに行ったり、えいがを見たりします。（星期天，我准备做一些事，比如去餐厅吃吃东西、看一看电影之类的。）

这是什么时态呢？

尽管我们使用的是动词的过去时态（た形），但整个句子的时态是由句尾的します决定的。

- 日曜日はレストランに行ったり、えいがを見たりします。（星期天，我会做一些事情，比如去餐厅吃吃东西、看一看电影之类的。）
- きのうはべんきょうをしたり、うんどうをしたりしました。（昨天，我做了一些事情，比如学习、运动之类的。）
- 来年は日本に行ったり、アメリカに行ったりしたいです。（明年，我想要去一去日本，或者去一去美国之类的地方。）


---

当我们谈论两件喜欢做的事时，我们用以下句型：
动词A的た形 + り、动词B的た形 + りすることが + すきです。

⚠️ 我们同样可以用 りするのが代替りすることが。

本を読んだり、りょうりをしたりするのが好きです。（我喜欢做一些事情，比如阅读和烹饪之类的。）

うたをうたったり、えいがを見たりすることが好きです。（我喜欢做一些事情，比如唱歌和看电影之类的。）

## 为什么
どうして可以用于询问“为什么”。
在构成这样的问句时，要把どうして 放在主语后面，如果主语被省略，就放在问句的开始部分。

どうしてですか 可以用在询问他人的对话中——为什么呢？

- どうして日本が好きですか。/ どうしてにほんがすきですか。（你为什么喜欢日本？）
- どうしてびょういんに行きますか。（你为什么要去医院？）
- どうしてですか。（为什么呢？）
